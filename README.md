This Power BI dashboard analyzes Prime Video content data to uncover trends across movies and TV shows. It converts raw streaming catalog data into interactive visual insights for content distribution, ratings, genres, and release patterns.

Key features of the dashboard include:
- Total titles, genres, ratings, and directors overview
- Content distribution by Movies vs TV Shows
- Rating category analysis
- Genre-wise title breakdown
- Country-wise content availability map
- Release year trends by content type
- Category and type-based filtering using slicers

This project demonstrates practical skills in data transformation, Power BI data modeling, DAX measures, and interactive dashboard design for media content analytics.
